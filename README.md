# フィギュア価格比較スクレイピングツール

メルカリとAmazonからフィギュアのデータをスクレイピングし、相場価格を比較するWebアプリケーションです。

## 機能

- 🔍 メルカリ・Amazon同時検索
- 💰 相場価格の自動計算（外れ値除外）
- 📊 サイト別価格比較
- ✏️ 商品情報の編集・削除
- 🎨 モダンなダークモードUI

## 技術スタック

- **フロントエンド**: React, Vite, Vanilla CSS
- **バックエンド**: Express.js
- **スクレイピング**: Playwright + Stealth Plugin

## セットアップ

### 1. 依存パッケージのインストール

```bash
npm install
```

### 2. Playwrightのブラウザをインストール

```bash
npx playwright install chromium
```

### 3. 開発サーバーを起動

```bash
npm run dev
```

これでフロントエンド（http://localhost:5173）とバックエンド（http://localhost:3001）が同時に起動します。

## 使い方

1. ブラウザで http://localhost:5173 にアクセス
2. 検索フォームにフィギュア名を入力
3. 検索対象のサイト（メルカリ/Amazon）を選択
4. 「検索」ボタンをクリック
5. 結果一覧と相場価格が表示されます

## 注意事項

⚠️ **利用規約について**
- メルカリ・Amazonの利用規約ではスクレイピングが禁止されている場合があります
- 本ツールは**個人利用・学習目的**でのみ使用してください
- 過度な使用はアカウント停止のリスクがあります

⚠️ **推奨使用頻度**
- 1日あたり50検索以下
- 各検索間に十分な間隔を空ける
